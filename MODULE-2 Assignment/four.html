4. List any 3 ways which help us in imporving the accessibility of html

1. Use Semantic HTML: Semantic HTML elements like
header nav main article footer
provide meaningful structure to your content. This helps screen readers and other assistive technologies understand and navigate your site more effectively.
2. Provide Alternative Text for Images: Always include descriptive alt attributes for images. This ensures that users who rely on screen readers can understand the content and purpose of the images on your site.
3.Ensure Keyboard Accessibility: Make sure all interactive elements (like links, buttons, and form controls) are accessible via keyboard. This includes using proper HTML elements and attributes to ensure users can navigate and interact with your site without a mouse.
