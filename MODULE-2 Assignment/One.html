1. List out the features of HTML5.

1. Syntax Highlighting: Provides color coding for HTML tags, attributes, and content, making it easier to read and debug code.
2. IntelliSense: Offers smart completions based on variable types, function definitions, and imported modules. This includes HTML tag suggestions and attribute completions.
3. Emmet Support: Allows for quick HTML and CSS code generation using abbreviations. For example, typing div.container>ul>li*5 and pressing Tab will expand to a structured HTML snippet.
4. Auto-closing Tags: Automatically closes HTML tags as you type, reducing the chance of errors.
5. Auto-update Tags: When you modify an opening tag, the corresponding closing tag is automatically updated.
6. Color Picker : A built-in color picker for editing colors in CSS within HTML files.
7. Hover Information : Displays information about HTML tags, attributes, and embedded CSS/JavaScript when you hover over them.
8. Validation : Performs validation on HTML, CSS, and JavaScript within HTML files to catch errors early.
9. Folding: Allows you to collapse and expand sections of your HTML code for better readability.
10. Formatting: Provides commands to format your HTML code, ensuring consistent and clean code structure.
