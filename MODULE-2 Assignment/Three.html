3. What is web accessibility? List some of the assistive devices which play a major role in providing accessibility.

Web accessibility ensures that websites, tools, and technologies are designed and developed so that people with disabilities can use them. This means that individuals can perceive, understand, navigate, and interact with the web, regardless of their disabilities.
Here are some key assistive devices that play a major role in providing accessibility:
1. Screen Readers: These are essential for individuals with visual impairments. They read aloud the text on a web page, helping users navigate and understand the content.
2. Magnification Tools: These tools enlarge on-screen content, making it easier for users with low vision to read and interact with web pages.
3. Speech Recognition Software: This software allows users to control their computers and input text using their voice, which is particularly useful for individuals who cannot use a keyboard or mouse.
4. Alternative Text (Alt Text) Generators: These tools provide text descriptions for images, which screen readers can then read aloud, making visual content accessible to those with visual impairments.
5. Captioning and Transcription Tools: These tools provide text versions of audio content, which are crucial for individuals with hearing impairments.
6.Keyboard Navigation Enhancements: These enhancements allow users to navigate websites using only their keyboard, which is helpful for those who cannot use a mouse.
