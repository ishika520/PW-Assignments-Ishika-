6. List any 5 semantic tags with their descriptions

1.Header: This tag defines the introductory content of a web page, typically containing the logo, navigation bar, and other introductory elements.
2. nav: This tag defines a section of the page that contains navigational links. This helps users and search engines understand how to navigate the website.
3. section: This tag defines a generic section of a document or a webpage. It groups related content
together and helps with content organization.
4. article: This tag defines a self-contained piece of content, like a blog post or news article. It can be reused independently on other pages.
5. aside: This tag defines content that is indirectly related to the main content of the page.
