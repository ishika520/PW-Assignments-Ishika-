What is HTML? Semantic HTML Common Tags

What is HTML?

HTML, short for HyperText Markup Language, is the building block of web pages. It defines the structure and content, using tags to create elements like headings, paragraphs, images, and more.

Semantic HTML Explained

Semantic HTML elevates web development beyond just structure. It utilizes tags that describe the meaning of the content. This enhances accessibility, search engine optimization (SEO), and overall code maintainability.

Common Semantic Tags

  • <header> - Introduces the webpage (logo, navigation)
  • <nav> - Defines a navigational section
  • <section> - Represents a generic content section
  • <article> - Defines self-contained content (news, blog post)
  • <aside> - Content indirectly related to the main content (sidebar)
