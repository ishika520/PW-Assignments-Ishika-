2. What are HTML Entities? List out 5 commonly used HTML Entities.

HTML entities are special codes used to represent characters that have a specific meaning in HTML, such as <, >, and &. These characters are reserved in HTML and must be replaced with entities to be displayed correctly in a web page.
Here are 5 commonly used HTML entities:
Less than: < ( Displays < )
Greater than: > (Displays >)
Ampersand: & (Displays &)
Double quotation mark: " (Displays ")
Non-breaking space:   (Displays a space that prevents line breaks)
