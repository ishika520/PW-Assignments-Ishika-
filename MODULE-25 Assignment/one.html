Q1. What is a Database? Explain with an example on why should we need a database?

Document A databaseis an organized collection of data stored and accessed electronically. Think of it as a digital filing cabinet, but instead of paper files, it stores information in a structured format that makes it easy to search, retrieve, and manage.

We need a database because:

Imagine you're running a small bookstore. You have information about books (title, author, price, quantity), customers (name, address, contact), and sales (date, book, customer). Without a database, you'd probably use spreadsheets or even pen and paper to manage this information. As your bookstore grows, this manual system becomes increasingly inefficient and error-prone.

A database solves this problem by:

Organizing data: It stores information in a structured way (like tables with rows and columns), making it easy to find specific data.
Efficiency: You can quickly search for information, generate reports, and analyze sales trends. Accuracy: Reduces errors in data entry and retrieval. Sharing: Multiple people can access and update the data simultaneously.
Security: Databases have built-in security features to protect sensitive information.
